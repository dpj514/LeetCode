## 6. ZigZag Conversion
将字符串转换成zigzag的形态，然后按照从左到右的顺序输出新的字符串，可设定方向和当前行序列，在转折点处改变方向，方向向上降低行序列，向下则相反。每行的字符进行拼接，最后拼接得到结果字符串。
